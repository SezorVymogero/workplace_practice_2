# Использованные промпты

## Общая информация

В ходе выполнения практического задания AI-19 для сравнения возможностей ChatGPT, Claude и Codex использовались одинаковые промпты. Это позволило обеспечить корректность сравнения и объективность полученных результатов.

# Промпт №1. Анализ технического задания

## Текст промпта

> Проанализируй следующее техническое задание. Необходимо разработать веб-приложение для управления задачами команды. (все в одном ответе)
>
> Требования:
>
> - регистрация и авторизация пользователей;
> - создание, редактирование и удаление задач;
> - назначение исполнителей;
> - изменение статуса задачи (Новая, В работе, Завершена);
> - поиск и фильтрация задач;
> - уведомления по электронной почте;
> - адаптивная версия для мобильных устройств.
>
> Необходимо:
>
> - выделить основные функциональные требования;
> - найти возможные риски и проблемы реализации;
> - определить недостающие детали и вопросы к заказчику;
> - предложить рекомендации по улучшению ТЗ.

### Назначение промпта

Оценка способности AI анализировать требования проекта, выявлять риски и формировать рекомендации по улучшению технического задания.

# Промпт №2. Генерация программного кода

## Текст промпта

> Напиши консольное приложение на Python для управления библиотекой книг. (все в одном ответе)
>
> Требования:
>
> - добавление книги;
> - удаление книги;
> - поиск книги по названию;
> - вывод списка всех книг;
> - сохранение данных в JSON-файл;
> - загрузка данных из JSON-файла при запуске программы.
>
> Дополнительные требования:
>
> - использовать объектно-ориентированный подход;
> - предусмотреть обработку ошибок;
> - добавить комментарии к основным частям кода.
>
> После того, как напишешь код, кратко объясни архитектуру решения.

### Назначение промпта

Оценка качества генерации программного кода, соблюдения требований и способности проектировать архитектуру приложения.

# Промпт №3. Code Review

## Текст промпта

> Выполни code review следующего кода. (все в одном ответе)
>
> Необходимо:
>
> - найти ошибки;
> - найти потенциальные проблемы;
> - указать нарушения хороших практик;
> - предложить улучшенную версию решения.
>
> Код:
>
> ```python
> users = []
>
> def add_user(name, age):
>     users.append({
>         "name": name,
>         "age": age
>     })
>
> def get_average_age():
>     total = 0
>
>     for user in users:
>         total += user["age"]
>
>     return total / len(users)
>
> def remove_user(index):
>     users.pop(index)
>
> add_user("Alex", 25)
> add_user("John", 30)
>
> remove_user(10)
>
> print(get_average_age())
> ```

### Назначение промпта

Оценка способности AI находить ошибки в коде, выявлять архитектурные проблемы, соблюдать хорошие практики программирования и предлагать качественные улучшения.
